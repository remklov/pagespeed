# Directory Overview

This directory contains PageSpeed Insights data for various websites, collected on November 3rd, 2025. The data is organized into subdirectories based on the Content Management System (CMS) used by each website.

## Key Files

The main contents of this directory are JSON files, each representing a Lighthouse performance report for a specific website. The files are organized as follows:

- **`Hubspot/`**: Contains reports for websites built with Hubspot.
- **`Ibexa/`**: Contains reports for websites built with Ibexa.
- **`Magento/`**: Contains reports for websites built with Magento.
- **`TYPO3/`**: Contains reports for websites built with TYPO3.
- **`WordPress/`**: Contains reports for websites built with WordPress.

The JSON files are named using the convention: `<domain-name>_<date>_<time>.json`.

## Usage

This directory is intended for storing and analyzing PageSpeed Insights data. The JSON files can be used to:

-   Track the performance of websites over time.
-   Compare the performance of websites built with different CMS platforms.
-   Identify common performance bottlenecks for each CMS.
-   Generate reports and visualizations of the performance data.
